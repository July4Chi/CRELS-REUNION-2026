# 🤝 CONTRIBUTING — 貢献ガイド

**Scope:** CRELS-REUNION-2026（コア）／ CRELS-REUNION-Extensions（拡張）共通

This research is open-source. 否定的な実験結果も、肯定的な結果と同じ価値を持つ貢献として歓迎します。

## Collaboration Protocol

1. **Fork repository**（GitHub: [CRELS-REUNION-2026](https://github.com/July4Chi/CRELS-REUNION-2026) / [CRELS-REUNION-Extensions](https://github.com/July4Chi/CRELS-REUNION-Extensions)）
2. **Conduct experiments** — 必ず安全プロトコルに従うこと（`ancient_tech/README.md` の「Safety First」参照。特にソテツはサイカシンを含むため取り扱い注意）
3. **Document findings** — 否定的な結果（null results）も貴重なデータとして記録する
4. **Submit pull request** with:
   - Detailed methodology（詳細な手順）
   - Photos / videos
   - Raw data（生データ）
   - Analysis（分析）
5. **Discuss in Issues tab** — 質問・アイデア共有・作業の調整

## 編集基準（Documentation Rules）

文書への貢献は **[EDITORIAL_STANDARD.md](EDITORIAL_STANDARD.md)** に従ってください：

- 事実レイヤーと仮説レイヤーを分離する
- 未検証の主張には【未検証仮説】ラベルを付す
- AI対話由来の評価値を「検証」として引用しない
- 用語は **[GLOSSARY.md](GLOSSARY.md)** の定義に合わせる
